Supabase Sync
Connect Supabase for shared data sync across all devices. Create a free project at supabase.com, then run the setup SQL below in the Supabase SQL editor.
CREATE TABLE crm_data ( id TEXT PRIMARY KEY, data JSONB, updated_at TIMESTAMPTZ DEFAULT NOW() ); ALTER TABLE crm_data ENABLE ROW LEVEL SECURITY; CREATE POLICY "allow_all" ON crm_data FOR ALL USING (true) WITH CHECK (true);
